💡 網頁綠色免安裝技術
本程式是一個「單檔案 HTML 網頁程式」。您不需要下載安裝軟體，只需在有網路的環境下載或直接打開線上網頁即可運作。所有學生的成績、名冊與分組均儲存在您目前電腦瀏覽器的暫存（LocalStorage）中，沒有任何伺服器會儲存您的資料，安全保密、讀取極速！

1 第一步：匯入班級資料

當您初次使用此程式時，請先匯入班級學生名冊。本程式支援讀取 Excel 檔案（如 .xlsx、.xls 及 .csv）。

📋 匯入按鈕對照
📋 開啟選取分頁 💾 儲存此分頁為班級 📚 匯入全部班級

（您會在「1. 匯入班級」分頁中看到這些按鈕）

詳細按鈕操作與欄位功能：

按鈕 / 欄位名稱 功能說明與操作教學
選擇 Excel 檔案
(上傳檔案)		 點擊以選取您電腦上的 Excel 名單。支援包含多個分頁（工作表）的檔案。
選擇班級分頁
(下拉選單)		 如果上傳的 Excel 擁有多個分頁，此處可下拉切換要解析的分頁。如果只有一頁，系統會自動選取。
匯入格式
(下拉選單)		 告訴系統如何解讀您的表格欄位：
  • 自動判斷座號與姓名（推薦）：會自動偵測「座號/姓名/學號」等標題。
  • 前兩欄為座號、姓名：強迫以表格最左側的第 1 欄當座號、第 2 欄當姓名。
  • 只匯入姓名欄位：僅提取姓名，不匯入座號。
📋 開啟選取分頁 臨時預覽所選分頁的學生名單，會直接將名單載入至「3. 上課選取」頁面。（注意：此動作僅供暫時測試，尚未存入網頁記憶體中）
💾 儲存此分頁為班級 將目前選擇的工作表解析後的名單正式儲存到瀏覽器的本地儲存空間（LocalStorage）中。以後只要打開網頁，即可在「選擇班級」中一鍵開啟。
📚 匯入全部班級 【整校/多班級匯入神器】 如果 Excel 當中每個分頁代表一個班級，點選此按鈕，系統會一次性把所有分頁全部儲存成不同的班級，省去手動一頁頁存檔的麻煩。
💡 智慧班級拆分技術
如果您的同一個 Excel 工作表裡面混合了多個班級的學生名冊（例如跨班選修），只要座號重新從 1 號開始，程式會自動辨識並分割為「工作表-班級1」、「工作表-班級2」等獨立班級。

2 第二步：選擇班級與備份管理

管理與切換您的班級資料庫。不論您要切換到下一堂課的班級、將資料備份匯出，或是刪除舊班級，皆在此處進行。

📂 班級資料管理
⬇️ 備份 JSON ⬆️ 還原 JSON ✨ 練習示範資料
🗑️ 刪除選取班級 🗑️ 刪除所有匯入班級

管理功能操作指南：

區塊與按鈕 操作說明與引導
選擇班級
(下拉選單)		 點擊後會顯示您在系統中儲存的全部班級與人數。選擇班級會立即載入該班的學生名單、隨機抽籤分數、個人評語以及所有的歷史分組紀錄。切換時系統會自動存檔前一個班級。
目前顯示班級
(唯讀文字框)		 顯示當前視窗中正載入的班級與狀態（例如：已儲存｜高一3班），供您隨時確認。
⬇️ 備份 JSON 將目前瀏覽器儲存的所有班級名單、加減分紀錄、備註與分組紀錄，打包成一個 .json 檔案下載儲存。建議定期備份，以防電腦重灌或瀏覽器被清理快取。
還原 JSON 備份檔
(上傳檔案與 還原 JSON)		 當您要在另一台教室電腦上課，或想要將資料還原時，在此處選擇您備份的 .json 檔案，點擊「還原 JSON」即可一鍵復原所有班級的全部資料。
✨ 練習示範資料 自動載入兩個虛擬班級（示範一班與示範二班），內建預設名單。方便您在沒有 Excel 檔案前，直接進行抽籤、分組等功能測試。
🗑️ 刪除選取班級 在右側下拉選單「選擇要刪除的班級」選定後，點選此按鈕以刪除單一班級的全部名冊、記錄與分組。
🗑️ 刪除所有匯入班級 【警告】 一鍵清除瀏覽器暫存中儲存的所有班級與成績，此操作不可逆，請務必先備份 JSON 檔。

3 第三步：上課選取與表現紀錄

課堂提問與點名抽籤的核心操作面板。支援「不重複隨機抽選」與「手動點選」雙模運作，並可針對表現進行即時評分與備註。

🎯 抽籤與評分控制台
🎲 開始隨機選取 🔄 清除選取 💾 儲存目前班級紀錄
👀 更新學生預覽 ✂️ 刪除學生模式 🗑️ 清空目前畫面

操作細節與實用技巧：

  • 手動輸入或微調名冊：在「學生名單」大文字框中，每行輸入一個名字（可加座號，如：01 王大同），修改後按 👀 更新學生預覽 即可反映在下方的名單卡片。
  • 刪除個別學生：點選 ✂️ 刪除學生模式，卡片會出現 × 按鈕，點擊可直接將該學生從目前名單中剃除。完成後再次點擊該按鈕即可退出刪除模式。
  • 點擊卡片手動點選：直接點擊學生名單中的卡片，選中後會變為 綠色，並在結果區最上方自動跳出該學生的評分列。這也是手動加減分最快的方法。
  • 🎲 隨機抽籤（不重複）：
    1. 在「選取學生人數」設定每次抽幾人（1~5人，或選其他自訂人數）。
    2. 在「選取次數」設定要抽幾組（例如設定 3 次，代表會一口氣連續做 3 組抽籤）。
    3. 點選 🎲 開始隨機選取，系統抽中的卡片會變綠色。
    4. 不重複機制：系統會自動記錄這輪被抽過的學生，並在下一次抽籤中主動避開他們，保證課堂公平。直到所有人抽過，或手動點選 🔄 清除選取，才會重新開始新一輪抽籤。

每次抽籤或點選後，結果區會產生一張張「結果卡片」：

欄位 / 元素 操作說明與引導
加減分控制鍵
[ − ] [ 0 ] [ + ]		 按 [ + ] 會使該學生的這項表現總分加 1，按 [ − ] 減 1。您也可以直接點進框內填寫分數（如：5 或 -2）。
快速備註選單
(下拉選單)		 點擊下拉選單，可以快速將常見表現語填入備註欄中：

🎤 語音輸入按鈕 【免打字錄音技術】 點擊麥克風按鈕（變為紅色 ⏹️），對著電腦麥克風講述該學生的表現（例：「上台解題條理清晰」），系統會自動將辨識出的文字填寫進備註欄中！

(結果大卡頂部輸入框)		 在頂部的文字框輸入「例：第1單元小考」、「抽問第3題」，該大卡中被選中的所有學生均會自動標註此共同事件。
操作（移除 / 刪除） 點擊單人右側的 移除 可將該生剔除本次選取卡片。點擊大卡右上角的 🗑️ 刪除 會完全將該次抽籤紀錄從系統中清空。

4 第四步：隨機小組分組安排 (全新功能)

★ 這是本次更新的核心功能！

課堂中需要分組報告、實驗或討論時，本分頁可隨機將學生拆分成不同組別，並可以對「小組整體」進行評分與事件備註。

🧩 隨機分組設定台
🎲 產生隨機分組 💾 儲存分組紀錄 🖨️ 列印分組名單 🗑️ 清除分組結果

分組參數設定詳解：

設定欄位 功能說明與使用方法
分組名稱
(文字輸入框)		 輸入此分組的主題，例如 第一次分組實驗 或 小組討論活動。若不輸入，系統會自動代入「分組 + 目前時間」。
分組方式
(下拉選單)		 選擇如何切分組別：
  • 依每組人數：指定「每組幾人」（如：4人），系統會依照目前人數除以 4 計算出組數並進行分組。（最後一組若人數不足，會自動分配剩餘的所有人）
  • 依組數：指定「分成幾組」（如：分成 4 組），系統會自動計算，將學生平均輪流分配到 4 個組別中。
分組來源
(下拉選單)		 設定要拿哪些人來分組：
  • 目前班級全部學生：最常用，對班級名單內的所有人進行完整分組。
  • 目前已選取學生：【智慧特定分組】 如果您在上課選取頁面中點選了部分同學（呈綠色卡片），在此選擇此項，將只對那些被選取的學生進行隨機分組，非常適合遲到同學分組或特定發表小組分組。
名單晶片預覽區
(分組頁面上方)		 顯示目前的班級名單。點擊名單中的名字可以「選取/取消選取」學生（綠色代表已選中），這會同步影響您的「分組來源」。

  • 小組表現備註：可以在小組卡片最下方的「輸入本組備註...」框中為該小組寫下評論（例如：「小組分工極佳」、「討論熱烈」）。
  • 儲存與列印：
    • 點擊 💾 儲存分組紀錄 可手動將分組資料存入系統中。
    • 點擊 🖨️ 列印分組名單，會開啟全新簡潔視窗，將本次所有組別的名冊表格化以供列印貼在公布欄。

✨ 特色功能與操作重點提示

1. 連鎖自動存檔 (Auto-Save) 技術

為了避免教師們在上課中因瀏覽器意外關閉或當機丟失心血，本程式在以下時機均會自動執行存檔：

  • 在上課選取頁面中點選學生、修改加減分、新增語音或快速備註時。
  • 在分組安排頁面中產生新分組、修改組分數、填寫小組備註時。
  • 切換頂部標籤分頁或點擊下拉選單切換不同班級時。
  • 當您要離開此網頁或關閉瀏覽器分頁時。

2. 如何利用「上課選取」與「隨機分組」進行連貫搭配

這是一個非常實用的教學情境搭配：

  1. 上課時，先在「3. 上課選取」頁面使用隨機抽取 8 位同學回答問題。
  2. 隨後，要進行實驗或小組研討。切換到「4. 分組安排」分頁。
  3. 將「分組來源」設定為 「目前已選取學生」，分組方式設定「依組數」分成 2 組。
  4. 點選隨機分組，系統便會直接把剛才這 8 位同學隨機均分成 2 組，進行特定的發表或研討。其他沒有被選中的同學則繼續保持，極其方便靈活！

3. 安全性提醒：定期下載 JSON 備份檔

雖然有自動存檔機制，但 LocalStorage（網頁暫存）是附屬於瀏覽器的。如果您的電腦被學校的硬碟還原系統卡鎖定、或您手動清理了瀏覽器 cookie 與快取，暫存資料會被抹除。
建議防範策略：每個禮拜五或下課前，去「2. 選擇班級」頁面點選 ⬇️ 備份 JSON 下載備份檔，這是最保險的成績管理習慣。

❓ 常見問與答 (FAQ)

Q: 在「分組安排」分頁進行隨機分組後，學生得到的「組分數」會加進「統計輸出」中的個人總分嗎？
不會。為了避免成績計算混淆，「上課選取（個人表現）」與「分組紀錄（小組整體分數）」兩者是分開記錄且各自獨立計算的。您可以分別在統計頁面切換兩種模式進行查看或下載各自的 Excel/PDF 文件。
Q: 為什麼隨機分組時，最後一組的人數跟其他組不一樣？
當您使用「依每組人數」進行隨機分組時，若班級總人數無法被設定的人數整除，就會產生餘數。例如：全班有 29 人，設定每組 4 人，系統會產生 7 組 (各4人)，並把剩下的 1 個人安排在最後的第 8 組。這屬於正常現象。
Q: 「列印分組名單」與「列印報表」有什麼不同？
  • 列印分組名單：位於「分組安排」分頁中。它是專門為了讓老師們能夠快速印出「乾淨的、不含加減分數與私密備註的純小組組員名單」，方便剪下來貼在教室或實驗室公佈欄。
  • 列印報表：位於「統計輸出」分頁。它是完整的平時成績與紀錄明細，含有分數、歷史抽籤時間與教師個人備註。
Q: 更換電腦上課時，要如何將班級、抽籤成績與分組資料一起帶走？
請在原本電腦上切換到「選擇班級」分頁，點擊「⬇️ 備份 JSON」下載檔案。上課時，在新的教室電腦打開本網頁，同樣去「選擇班級」分頁，選擇該 JSON 檔案，點選「⬆️ 還原 JSON」，名單、所有成績和分組便會一秒全部回復。
