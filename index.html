學生選取與紀錄程式
教師極簡化操作指南

這是一套專為中小學教師開發的「課堂點名抽籤、加減分與課堂表現即時紀錄」工具。無需繁雜的資料庫設定，只需打開網頁，即可快速開始您的智慧教學。

工具版本：v2026.06 優化版 技術特點：純網頁免安裝、在地隱私、Excel相容、語音辨識

💡 1分鐘快速入門

為了讓您在課堂上游刃有餘，本程式設計了極為直覺的四步工作流。請對照程式頂部的標籤切換頁面：

1

匯入班級

拖入 Excel 檔案，讀取分頁建立學生名冊

2

選擇班級

下拉選單一鍵切換，支援多個班級同時保存

3

上課點選

進行隨機抽籤或手動點選，輸入分數與評語

4

統計輸出

查看累計分數，下載 Excel/PDF 報表歸檔

💡 溫馨提示（免下載、免安裝）
本程式是一個「單檔案 HTML 應用網頁」。只要將此網頁存在隨身碟或電腦裡，點擊兩下就能直接在瀏覽器執行。所有資料皆儲存在您本機的網頁暫存中，不會上傳到任何外部伺服器，絕無學生隱私洩露風險！

1 第一步：匯入班級資料 (匯入分頁)

當您第一次使用，或者新學期剛開始時，您需要在此頁面匯入學生名單。系統支援直接讀取 Excel 檔案（如 .xlsx、.xls 等格式）。

📋 介面按鈕對照
📋 開啟選取分頁 💾 儲存此分頁為班級 📚 匯入全部班級

（您會在「1. 匯入班級」分頁中看到上述按鈕）

按鈕功能與詳細操作說明：

按鈕名稱與位置 功能操作與引導
選擇 Excel 檔案
(檔案上傳輸入框)		 點擊並選擇您的班級學生名冊 Excel 檔。若您的 Excel 中包含多個班級（例如一個分頁一個班級），程式會完整載入並讓您自由選取。
選擇班級分頁
(下拉選單)		 若 Excel 中有多個工作表（Sheets），此處可選擇要讀取的工作表名稱。若只有一個，系統會自動選取。
匯入格式
(下拉選單)		 設定系統如何分析您的 Excel 欄位：
  • 自動判斷座號與姓名：最推薦，會自動掃描包含「座號、姓名、學生」等關鍵字欄位。
  • 前兩欄為座號、姓名：若您的表格很單純，最左邊兩欄就是座號與姓名，選此項。
  • 只匯入姓名欄位：不匯入座號，僅匯入學生姓名。
📋 開啟選取分頁 【立即載入測試】 點擊此按鈕，系統會解析目前選取的工作表，並將學生名單直接顯示在「3. 上課選取」頁面。（注意：此操作尚未將資料存入瀏覽器的記憶體中，適合臨時使用）
💾 儲存此分頁為班級 將目前選擇的工作表，儲存到瀏覽器的本地儲存空間 (LocalStorage) 中。往後您只要打開此網頁，不需要再次載入 Excel，即可直接在「2. 選擇班級」中開啟該班。
📚 匯入全部班級 【大批匯入神器】 如果您的 Excel 檔案包含全校或多個班級（一頁一班），點擊此按鈕，程式會一次性把所有分頁全部存入系統記憶體中，非常省時省力。
💡 跨班級自動拆分功能
如果您的 Excel 當中同一個分頁包含多個班級的座號與姓名，程式具備「智慧拆分技術」，會自動偵測座號是否重新從 1 號開始，並自動將其拆分為「工作表-班級1」、「工作表-班級2」等獨立班級。

2 第二步：選擇班級與備份管理

當您匯入或儲存了多個班級後，您可以在此頁面自由切換目前上課的班級、進行資料備份下載，或是清除不必要的舊名單。

📂 班級管理與 JSON 備份
⬇️ 備份 JSON ⬆️ 還原 JSON ✨ 練習示範資料
🗑️ 刪除選取班級 🗑️ 刪除所有匯入班級

功能與詳細操作說明：

操作區塊 / 按鈕 功能操作與引導
選擇班級
(下拉選單)		 列出目前所有已儲存的班級。選擇一個班級後，系統會自動切換，並即時載入該班的歷史評分紀錄。切換班級時，前一個班級的變更會自動被儲存。
目前顯示班級
(唯讀狀態欄)		 顯示當前畫面上正在查看的班級狀態。例如 已儲存｜高一1班，讓您隨時確認沒有選錯班級。
⬇️ 備份 JSON 【定期備份防丟失】 將所有班級名單、所有的加減分與評語紀錄，整合成一個 .json 檔案下載到您的電腦中。這能防止瀏覽器快取被清理時導致資料不見。
JSON 備份檔 / 還原
(檔案欄位與 還原 JSON)		 當您需要換電腦上課，或想要將資料還原到先前的狀態，在此處選擇您備份的 .json 檔案，點擊「還原 JSON」即可一鍵還原所有設定。
✨ 練習示範資料 如果是第一次使用，點擊此按鈕，系統會自動產生「示範一班」與「示範二班」的虛擬名單，方便您先熟悉功能，不用急著找 Excel。
🗑️ 刪除選取班級 在「選擇要刪除的班級」選單選定後，點擊此按鈕可刪除該單一班級的全部名單與紀錄。
🗑️ 刪除所有匯入班級 清空瀏覽器中儲存的所有班級與所有分數紀錄。適合在每學年結束時進行系統大掃除。

3 第三步：上課選取與評分紀錄

這是您課堂上使用最頻繁的主畫面。本頁面提供「隨機抽籤（不重複）」與「手動點選」兩種模式，並可對選中學生進行即時打分與備註。

🎯 上課抽籤控制台
🎲 開始隨機選取 🔄 清除選取 💾 儲存目前班級紀錄
👀 更新學生預覽 ✂️ 刪除學生模式 🗑️ 清空目前畫面

核心操作功能介紹：

  • 手動名單編輯：在「學生名單」文字框中，您可以手動增加或修改名單，一行打一個名字（例如：01 王小明）。編輯後點擊 👀 更新學生預覽 就會更新名單卡片。
  • 手動點名模式：點選下方的學生名字卡片，選中後會變為 綠色。同時會在最下方立即彈出該名學生的加減分控制卡。
  • 不重複隨機抽籤：
    1. 在「選取學生人數」選取每次抽幾個人。
    2. 在「選取次數」設定要連續抽幾組。
    3. 點擊 🎲 開始隨機選取。
    4. 系統會自動避開這輪已經被抽中的學生。直到名單所有人都被抽過，或點擊 🔄 清除選取，才會重啟新一輪。

即時評分與記錄卡片：

每次選取（隨機或手動）後，最下方會列出選中的學生名冊，供您填寫課堂表現：

欄位 / 操作元素 如何使用與操作技巧
加減分控制
[ − ] [ 0 ] [ + ]		 點擊 [ + ] 會使該學生總分加 1 分，點擊 [ − ] 減 1 分。您也可以直接在框中輸入任何自訂數字（如：5）。
備註（手動輸入） 直接點擊輸入框，輸入這堂課他的具體表現（如：回答第一題、上台發表、未帶課本）。
快速備註選單
(下拉選單)		 點擊下拉選單，可以一鍵代入常用課堂事件評語：
表現良好 積極發言 協助同學 上課遲到 睡覺吵鬧 作業缺失
🎤 語音輸入按鈕 【課堂神速紀錄】 點擊麥克風按鈕，瀏覽器會開始聆聽。直接說出您的評語（例如：「發言積極，答案正確」），系統會自動將語音辨識為文字填入備註欄！
※ 請在安靜環境下使用，且首次使用需允許瀏覽器取得麥克風權限。
本次選取批次備註
(結果卡片頂部輸入框)		 可以設定這整批被抽到學生的共同事件（例如：「分組報告第一組」、「實驗操作」），整批學生會共享此事件背景。
操作（移除 / 刪除）
  • 點擊表格右邊的 移除：可以只把這名學生從這次的抽籤評分卡中拿掉。
  • 點擊右上角的 🗑️ 刪除：會把這整批（次）的抽籤卡片及紀錄完全刪除。
💾 自動存檔機制與手動存檔
當您在評分卡中調整分數、修改評語，或是切換頁面時，程式會自動儲存變更。 另外，您也可以在下課前隨時點擊 💾 儲存目前班級紀錄 確保所有更新安全寫入瀏覽器記憶體中。

4 第四步：加減分統計與報表輸出

在學期末或需要送出平時成績時，切換至本頁面。系統會自動為您計算每位學生的加減分總和、被選取次數，並提供導出功能。

📊 報表下載與列印
📊 下載 Excel 紀錄 📄 下載 PDF 報表 🖨️ 列印報表

報表輸出選項說明：

匯出格式與按鈕 格式內容與適用場景
📊 下載 Excel 紀錄 會導出一個包含三個 Sheet 的工作簿：
  1. 學生名單：目前班級的名冊與是否被選取狀態。
  2. 選取與個人紀錄：每次抽籤的時間、批次備註、加減分與個人評語。
  3. 加減分統計：每名學生的被抽中次數、加分累計、扣分累計、總得分與最近一次表現。
適合拿來做成績試算與二次加工。
📄 下載 PDF 報表 將整個紀錄表與統計表排版渲染成精美的 PDF 格式下載。適合用於繳交學校行政單位、列印出來貼在教室，或是寄給家長看學生的課堂平時表現。
🖨️ 列印報表 直接開啟系統的列印對話框，可將報表直接輸出至實體印表機，或使用系統內建的虛擬印表機存為 PDF。
📊 統計摘要卡片解讀
統計頁面上方有四張快速卡片：
  • 目前學生數：確認班級名單總人數。
  • 紀錄筆數：本學期課堂抽籤被評分的總人次。
  • 全班總分：本班目前累積的平時表現分數總和（常用於班級競賽或整體表現評估）。
  • 目前最高：顯示當前得分最高的學生姓名與分數。

✨ 特色功能專題解說

1. Excel 多班級「一鍵全部匯入」格式推薦

如果您的學校名單 Excel 包含多個分頁，例如「高一1班」、「高一2班」...，您可以直接整理成以下標準格式：

  • 第一欄（A欄）：座號 (可填寫 1、2、3 或 01、02...)
  • 第二欄（B欄）：姓名 (學生的真實姓名)

上傳該 Excel 檔後，直接點選 📚 匯入全部班級，程式便會自動幫您建立好所有的班級與名單。您只需要去「選擇班級」下拉選單切換，即可在不同的班級間快速切換。

2. 麥克風語音輸入 (Web Speech API)

在上課選取學生的評語欄位旁邊，設有 🎤 按鈕。這是利用瀏覽器內建的 Speech-to-Text 引擎，支援中文辨識。

📢 啟用步驟：
  1. 點擊 🎤 按鈕，此時按鈕會變成紅色 ⏹️，表示正在錄音。
  2. 朝向您的麥克風說話（如：「發言積極主動」）。
  3. 再次點擊 ⏹️ 停止，或靜待 2 秒，語音辨識的文字即會自動出現在備註欄的文字尾端。

3. 資料安全性與儲存機制

本程式的所有資料皆儲存於您目前使用的瀏覽器 LocalStorage（本機儲存區）。

  • 優點：上課時即使沒有網路，依然可以完全正常操作、評分、儲存。
  • 缺點：如果您清除瀏覽器的歷史紀錄與快取資料，或是換另外一台電腦，資料將無法共用。
  • 安全策略：建議教師們每週或每月，到「選擇班級」頁面點擊 ⬇️ 備份 JSON，下載一個小小的備份檔存在隨身碟或雲端雲碟，萬一瀏覽器快取被清理，也能在一秒內還原。

❓ 常見問與答 (FAQ)

Q: 網頁關閉後，我之前打的加減分和備註會消失嗎？
不會。只要您是在同一台電腦的同一個瀏覽器開啟，程式會自動將紀錄存放在瀏覽器的 LocalStorage 中。但為了安全起見，建議下課前可以點擊「儲存目前班級紀錄」或下載 JSON 備份。
Q: 點擊「語音輸入 🎤」按鈕時，為什麼沒有反應或顯示錯誤？
語音辨識功能需具備以下三個條件：
  • 使用的是支援語音功能的瀏覽器（強烈推薦使用 Google Chrome 或 Microsoft Edge 瀏覽器）。
  • 您的電腦已連接可正常運作的麥克風。
  • 瀏覽器彈出「要求麥克風權限」時，您必須點擊「允許」。若不小心點到拒絕，需到網頁網址列最左側的「鎖頭」圖示中重新開啟麥克風權限。
Q: 匯入 Excel 檔案時，畫面沒有任何動作？
可能原因有二：
  • 網路中斷：本程式在初次載入時，會從網路下載安全解析 Excel 的元件。如果您的電腦完全處於斷網狀態下第一次開啟本說明網頁，可能無法解析。請在有網路的環境下整理一次網頁後再使用。
  • Excel 格式太複雜：如果表格內含過多的合併儲存格或複雜公式，可能導致元件解析失敗。請將名單整理成最單純的兩欄（A欄座號、B欄姓名）格式再試一次。
Q: 如何把這台電腦的紀錄轉移到另外一台教室電腦？
非常簡單：
  1. 在原本的電腦，切換到「選擇班級」分頁，點擊 ⬇️ 備份 JSON，下載備份檔。
  2. 將備份檔（.json）透過隨身碟或雲端寄到另一台電腦。
  3. 在另一台電腦打開本程式，至「選擇班級」點選「選擇檔案」上傳該 JSON 檔，並點擊 ⬆️ 還原 JSON 即可。
